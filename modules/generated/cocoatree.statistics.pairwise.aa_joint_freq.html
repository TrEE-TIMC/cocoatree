cocoatree.statistics.pairwise.aa_joint_freq¶

cocoatree.statistics.pairwise.aa_joint_freq(sequences, weights, lambda_coef=0.03)¶

Computes the joint frequencies of each pair of amino acids in a MSA

f_{ij}^{ab} = (1 - \lambda) \sum_s w_s \frac{x_{si}^a x_{sj}^b}{M'} + \frac{\lambda^2}{(21)^2}

where

M' = \sum_s w_s

represents the effective number of sequences in the alignment and lambda is a small regularization parameter (default=0.03).

Arguments¶

sequences : list of sequences as imported by load_MSA()

weightsndarray of shape (Nseq)

sequence weights as calculated by seq_weights()

lambda_coeffloat

regularization parameter lambda (default=0.03)

Returns¶

joint_freqsndarray of shape (Nseq, Nseq)

amino acid joint frequencies

joint_freqs_indndarray of shape (Nseq, Nseq)

amino acid joint frequencies if independent
